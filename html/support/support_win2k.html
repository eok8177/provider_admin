Настройка соединения в Windows NT, 2000, XP

Для подключения к Internet вам необходимы:

  • Компьютер
  • Установленный модем
  • Создать удаленное соединение
  • Настроить удаленное соединение
  • Установить связь

Создание удаленного соединения

  1. Нажмите Пуск -> Настройка -> Сеть и удаленный доступ к сети
  2. Далее нажмите Создание нового подключения
  3. В появившемся Мастере сетевого подключения нажмите Далее -> Телефонное подключение по частной сети -> Далее.
  4. Выберите модем с помощью которого будет подключение и нажмите Далее
  5. В поле Номер телефона, введите телефон 77181 (перед номером телефона поставьте латинскую букву "P" - для импульсного набора). Нажмите Далее
  6. Введите название соединения (например, LokalNET) и нажмите Готово.

Настройка удаленного соединения

  1. Перейдите на вкладку Пуск -> Настройка -> Панель управления -> Сеть и удаленный доступ к сети.
  2. Щелкните правой кнопкой мышки на созданном соединении и в появившемся меню нажмите Свойства.
  3. Далее выберите закладку Сеть.
  4. Проверьте наличие галочки напротив пункта TCP/IP. Если он отсутствует - поставьте ее.
  5. Дважды щелкните мышкой на TCP/IP. Поставьте галочки в строках Получить IP-адрес автоматически и Получить адрес DNS-сервера автоматически. Нажмите OK.
  6. Далее выберите закладку Безопасность.
  7. Поставьте галочку в строке Вывести окно терминала.

Установка связи

  1. Перейдите на вкладку Пуск -> Настройка -> Панель управления -> Сеть и удаленный доступ к сети.
  2. Дважды щелкните левой кнопкой мышки на созданном вами соединении.
  3. В поле Имя пользователя введите Ваш логин, в поле Пароль Ваш пароль. Нажмите Вызов.
  4. В появившемся терминале (чёрное окно) введите Ваш логин и пароль. Отобразится состояние Вашего счета. Нажмите Готово
