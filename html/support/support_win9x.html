Настройка соединения в Windows 9x, ME

Для подключения к Internet вам необходимы:

  • Компьютер
  • Установленный модем
  • Установленный контроллер удаленного доступа и протокола TCP/IP
  • Установленный и настроенный удаленный доступ к сети
  • Установить связь

Установка контроллера удаленного доступа и протокола TCP/IP

  1. Установите инсталляционный диск Windows 95/98 в дисковод CD-ROM.
  2. Откройте My Computer (Мой компьютер) на рабочем столе Windows.
  3. Откройте Панель управления (Control Panel).
  4. Запустите Сеть (Network).
  5. В открывшемся окне нажмите кнопку Добавить (Add).
  6. В списке компонентов выберите Контроллер (Adapter) и нажмите Добавить (Add).
  7. В списке Производитель (Manufacturer) выберите Microsoft.
  8. В списке Сетевые адаптеры (Network adapters) выберите Контроллер удаленного доступа (Dial-Up Adapter), нажмите OK.
  9. В окне настройки сети появится иконка Контроллера удаленного доступа (Dial-Up Adapter).
  10. Снова нажмите кнопку Добавить (Add).
  11. В списке компонентов выберите Протокол (Protocol) и нажмите Добавить (Add).
  12. В списке Производитель (Manufacturer) выберите Microsoft.
  13. В списке Сетевые протоколы (Network protocols) выберите TCP/IP и нажмите кнопку ОК.
  14. Программа установки запросит инсталляционный диск Windows 95/98 и скопирует с него необходимые для установки файлы.
  15. Перезагрузите компьютер.

Установка удаленного доступа к сети

  1. Установите инсталляционный диск Windows 95/98 в дисковод CD-ROM.
  2. Откройте My Computer (Мой компьютер) на рабочем столе Windows.
  3. Если в открывшемся окне вы обнаружите иконку Удаленный доступ к сети (Dial-Up Networking), смотрите дальше "Настройка удаленного доступа к сети".
  4. Откройте Панель управления (Control Panel).
  5. Запустите Добавить удалить программы (Add/Remove programms).
  6. Нажмите на закладку Установка Windows (Windows Setup).
  7. В списке компонентов выберите Связь (Communications) и нажмите кнопку Details.
  8. Поствьте галочку возле Удаленного доступа к сети (Dial-Up Networking) и нажмите кнопку OK.
  9. Программа установки запросит инсталляционный диск Windows 95/98 и скопирует с него необходимые для установки файлы.
  10. Перезагрузите компьютер.

Настройка удаленного доступа к сети

  1. Откройте My Computer (Мой компьютер) на рабочем столе Windows.
  2. Откройте папку Удаленный доступ к сети (Dial-Up Networking).
  3. Запустите программу Создать новое соединение (Make New Connection).
  4. Введите название соединения LokalNET или любое другое по вашему вкусу. Выберите модем, установленный на вашем компьютере.
    5. Нажмите Далее (Next).
  5. Введите Код города (Area code) - 39 (Гопри), введите номер телефона, p77181, выберите страну - УКРАИНА. Нажмите кнопку Далее (Next).
  6. Надмите кнопку Готово (Finish).
  7. Щелкните правой кнопкой мышки на только что созданную иконку соединения. Из появившегося меню выберите Свойства (properties).
  8. Снимите галочку возле пункта Использовать код страны и города (Use area code:).
  9. Нажмите на закладку Тип сервера (Server types) оставьте галочку только возле пункта TCP/IP.
  10. Нажмите ОК.

Установка связи

  1. Нажмите два раза на иконку только что созданного соединения.
  2. Введите имя пользователя (Login) и пароль (Password), указанные на карточке.
  3. Нажмите кнопку Соединить (Connect).
  4. Дождитесь соединения.
