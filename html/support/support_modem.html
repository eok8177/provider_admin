Для Windows 98, ME

    Для начала инсталляционного процесса включите компьютер и загрузите Windows 95/98. Убедитесь в том, что модем включен и подключен к телефонной линии. Если модем еще не инсталлирован в Windows 95/98, то проведите инсталляцию.
1) Установите модем в компьютер следуя инструкции которая поставляется с модемом.
2) Откройте Мой Компьютер->Панель управления->Модемы->Добавить
3) Следуя инструкциям Windows 95/98 инсталлируйте модем.

Для проверки работы модема:
  Откройте Мой Компьютер->Панель управления->Модемы
  Выберите порт на котором установлен модем и нажмите "Дополнительно"
  На все посланные команды должен быть получен отклик.

Для Windows 2000

1) Откройте Мой Компьютер->Панель управления->Телефон и Модем
2) Подождите, пока программа установки обнаружит и установит модем. Нажмите "Далее".
3) Укажите код города и тип набора номера. Нажмите "Далее".

Проверить работу модема можно так:
  Откройте Мой Компьютер->Панель управления->Телефон и Модем.
  На вкладке Модемы выберете модем и нажмите кнопку Свойства
  На вкладке Диагностика выберите -> Опросить модем
  На все посланные команды должен быть получен отклик.
